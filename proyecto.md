# Recetario
## Manual Maestro de Cocina para Instant Pot

**Versión:** 1.0  
**Framework:** Laravel 13  
**Frontend:** Livewire + Alpine.js + TailwindCSS  
**Base de Datos:** MySQL / MariaDB  
**Objetivo:** Crear el mejor manual de recetas y técnicas culinarias para Instant Pot, enfocado en enseñar **cómo cocinar**, no únicamente **qué cocinar**.

---

# Filosofía del proyecto

La mayoría de los recetarios muestran únicamente ingredientes y pasos.

Este proyecto busca algo distinto.

El usuario debe comprender:

- Por qué se realiza cada paso.
- Qué ocurre durante la cocción.
- Cómo adaptar una receta.
- Qué errores evitar.
- Cómo utilizar correctamente la Instant Pot.

Al finalizar el manual, el usuario deberá ser capaz de crear sus propias recetas.

No será solamente un recetario.

Será un curso completo de cocina utilizando Instant Pot.

# Estilo de escritura

Todo el contenido deberá seguir un formato uniforme.

Nunca utilizar expresiones como:

❌

"Yo hago..."

"Mi receta favorita..."

"Yo recomiendo..."

Siempre utilizar:

✅

"La opción más recomendable..."

"Fundamento técnico..."

"Resultado esperado..."

"Objetivo..."

"Observaciones..."

El lenguaje deberá ser claro, preciso, didáctico y técnico.

---

# Objetivo final

Construir el mejor manual de Instant Pot disponible en español.

No solamente un recetario.

Un recurso de aprendizaje donde cada receta enseñe una nueva técnica culinaria y permita al usuario comprender el funcionamiento de la cocina bajo presión para posteriormente adaptar cualquier receta tradicional a la Instant Pot con criterio técnico.
